# Support 🆘

Thank you for using **Sadia AI**! If you're having trouble, here are the best ways to get help:

### 💬 Community Support (Recommended)
For setup questions, configuration help, or general talk, please use our **[GitHub Discussions](https://github.com/maheerCodes/Email-Automation/discussions)**. This helps others with the same problem find the solution.

### 🐛 Bug Reports & Feature Requests
If you found a bug or have a great idea, please open an **[Issue](https://github.com/maheerCodes/Email-Automation/issues)** using our templates.

### 🛡️ Security Issues
For security-related concerns, please refer to our **[Security Policy](https://github.com/maheerCodes/Email-Automation/security/policy)**.

---
*Note: Please do not open issues for general setup help; use Discussions instead!*
