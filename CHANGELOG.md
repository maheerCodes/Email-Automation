# Changelog

All notable changes to this project will be documented in this file. The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/) and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.1] - 2024-05-16
### 🚀 Added
- **Production Ready Release:** Full autonomous workflow for Sadia AI.
- **AI Inference:** Integrated **Google Gemini 1.5 Flash** for high-speed, intelligent replies.
- **RAG Architecture:** Grounded knowledge retrieval from **Google Sheets** to provide accurate service & pricing info.
- **Linguistic Fluidity:** Added native support for **English, Bangla, and Banglish** mirrored responses.
- **Contextual Intelligence:** Implemented **Thread-ID based memory** for coherent conversation flow.
- **Loop Protection:** Added smart filtering to prevent the AI from replying to itself.

### 🛠️ Changed
- Optimized n8n node structures for better error handling and lower latency.
- Refined the System Prompt for a more professional executive assistant persona.
