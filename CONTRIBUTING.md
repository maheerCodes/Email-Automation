# 🤝 Contributing to Sadia AI

First off, thanks for taking the time to contribute! ❤️

We love your input! We want to make Sadia AI the best autonomous assistant, and we need your help—whether it's fixing bugs, improving the RAG logic, or suggesting new nodes.

### 🚀 How Can I Contribute?

#### Reporting Bugs
- Check the [Discussions](https://github.com/maheerCodes/Email-Automation/discussions) to see if the bug was already reported.
- If not, open a new **Bug Report** issue.

#### Suggesting Enhancements
- Open a new **Feature Request** issue or start a **Discussion**.

#### Pull Requests
1. Fork the repo and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. Ensure the `.json` workflow file is clean and updated.
4. Issue that pull request!

### 📝 Style Guide
- Keep n8n node names clear and descriptive.
- If you use the Code Node (JavaScript), keep the code clean and commented.
- Always mirror the project's professional tone in system messages.

**Thank you for your contribution!**
